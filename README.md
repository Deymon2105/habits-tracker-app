# Habits Tracker App

Una aplicación web moderna y dinámica para el seguimiento de hábitos, construida con React 19, Tailwind CSS v4 y Supabase. Esta aplicación permite a los usuarios gestionar sus hábitos diarios y semanales de manera intuitiva y con una interfaz atractiva.

## Tecnologías Utilizadas

Este proyecto utiliza un stack tecnológico que garantiza un buen rendimiento y experiencia de usuario:

- **Frontend Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Componentes UI:** [HeroUI](https://heroui.com/)
- **Navegación:** [React Router DOM](https://reactrouter.com/)
- **Iconos:** [Lucide React](https://lucide.dev/)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
- **Backend / Base de Datos:** [Supabase](https://supabase.com/)

## Características Principales

- **Dashboard Interactivo:** Vista general del progreso de tus hábitos.
- **Vistas Semanales y Diarias:** Organiza y visualiza tus hábitos por día o por semana.
- **Gestión de Hábitos (CRUD):** Crear, leer, actualizar y eliminar hábitos fácilmente.
- **Seguimiento de Progreso:** Visualización clara del cumplimiento de objetivos.
- **Diseño Responsivo:** Funciona perfectamente en dispositivos móviles y de escritorio.

## Estructura del Proyecto

```
src/
├── assets/        # Imágenes y recursos estáticos
├── components/    # Componentes reutilizables
│   ├── layout/    # Header, Footer, Layout principal
│   └── ui/        # Componentes base de UI
├── pages/         # Páginas principales (Dashboard, WeekView, DayView)
├── services/      # Lógica de integración con API (Supabase)
├── styles/        # Estilos globales
└── App.jsx        # Componente raíz y configuración de rutas
```
